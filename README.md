# 🎮 Tic-Tac-Toe AI

An intelligent Tic-Tac-Toe game built using **HTML, CSS, and JavaScript** where the player competes against an unbeatable AI powered by the **Minimax Algorithm**.

---

## 📌 Project Overview

This project implements an AI agent that plays the classic game of Tic-Tac-Toe against a human player.
The AI uses the **Minimax Algorithm**, a popular decision-making algorithm in game theory, to always make the best possible move.

The project demonstrates:

* Artificial Intelligence basics
* Game Theory concepts
* Search Algorithms
* Minimax strategy implementation
* Interactive web development using JavaScript

## 🧠 AI Algorithm Used

### Minimax Algorithm

The AI uses the **Minimax Algorithm** to evaluate all possible future moves and choose the optimal move.

### How It Works

* The AI checks all possible game outcomes.
* It maximizes its own winning chances.
* It minimizes the player's winning chances.
* The AI never loses.

## 🎯 Game Rules

* Player uses **X**
* AI uses **O**
* First player to align 3 symbols horizontally, vertically, or diagonally wins.
* If all cells are filled without a winner, the game ends in a draw.
